## Annotation

*EE2MesGO is rewritten and updated version* [EE2E C++](https://github.com/angryduck1/E2EMes)

## Overview

E2EMes is a messenger with continious end-to-end encryption.

- End-To-End encryption is a main act of security
- Fast working and easy for use
- Comfortable session system of authorization.

**REST API** is use for authorization.
After authorization upgrades to **WebSocket**.

## Install

```
go mod download
go build
```
